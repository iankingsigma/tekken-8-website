## Brainrot Fighters

A fighting game with brainrot mechanics, featuring:
- Multiple characters with unique abilities
- Boss fights and special mechanics
- Shop system with upgrades
- Touch controls for mobile/tablet
- Parry system with cooldown

### Website: https://iankingsigma.github.io/tekken-8-website/

### Controls:
- **Arrow Keys**: Move left/right
- **Z/X**: Punch attacks
- **A/S**: Kick attacks
- **Space**: Parry (heals 10% HP, deals 20% damage, 10s cooldown)
- **C**: Special combo attack

### Boss Music:
- When fighting the 67 BOSS, the game will automatically play "Finale.mp3"
