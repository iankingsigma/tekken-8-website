  • NERFED PARRY: Now only deals 2% damage (no healing)
  • Added 10-second cooldown - button becomes unusable

  • Added Insane Mode - CPU parries 100% of attacks

  • Fixed shop display issues

  • Added CPU memory system - CPU learns from your moves

  • Added parry system - CPU has 50% chance to parry attacks
  • Added shop with purchasable items
  • Fixed tablet controls
  • Added forward/backward movement in 3D space
  • Added scrollable controls menu
  • Added credits screen
  • Removed versus mode

v3.0.0 - MAJOR UPDATE
